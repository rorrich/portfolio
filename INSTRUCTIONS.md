# Инструкция по запуску проекта

## Что это?
Это портфолио-сайт с анимированными элементами и переключением тем.

## Как запустить?

### Вариант 1: Через Python (рекомендуется)
1. Откройте терминал в папке проекта
2. Выполните команду: `python3 -m http.server 8001`
3. Откройте браузер и перейдите по адресу: `http://localhost:8001`

### Вариант 2: Через любой другой локальный сервер
- Live Server (VS Code)
- XAMPP
- MAMP
- Или любой другой локальный сервер

## Структура проекта
- `index.html` - главная страница
- `css/styles.css` - все стили
- `js/main.js` - JavaScript функциональность
- `images/` - все изображения и видео
- `mcp.json` - конфигурация для Figma плагина

## Особенности
- Переключение тем (кнопка "какой ты сегодня")
- Анимированные секции
- Адаптивный дизайн
- Интерактивные элементы

## Требования
- Любой современный браузер
- Локальный сервер (не открывать файл напрямую)
