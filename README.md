# Портфолио

Адаптивный сайт-портфолио, разработанный на основе дизайна из Figma.

## Описание

Проект представляет собой одностраничный сайт-портфолио с адаптивным дизайном. Страница включает:

- Шапку с логотипом и меню
- Героический блок с заголовком
- Секцию "Обо мне"
- Бегущую строку с навыками
- Раздел с работами
- Футер с контактами

## Функциональность

- Адаптивный дизайн для разных размеров экрана
- Анимации при наведении и скролле
- Смена цветовой схемы при клике на "какой ты сегодня"
- Интерактивные карточки работ

## Технологии

- HTML5
- CSS3 (flexbox, grid, анимации)
- JavaScript (без фреймворков)
- Шрифт Onest от Google Fonts

## Запуск проекта

1. Клонировать репозиторий:
```
git clone [url-репозитория]
```

2. Открыть проект в браузере:
   - Вариант 1: Открыть файл index.html напрямую
   - Вариант 2: Запустить локальный сервер (рекомендуется):
     ```
     python -m http.server 8000
     ```
     Затем перейти по адресу http://localhost:8000

## Особенности реализации

- Бегущая строка с навыками реализована с помощью CSS-анимаций
- Эффект появления элементов при скролле реализован с использованием Intersection Observer API
- Смена цветовых схем работает через атрибуты data-* и CSS-переменные
